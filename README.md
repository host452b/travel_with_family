## 量化指标 (0-10 分)

| 指标 | 0分 | 10分 | 权重 |
|------|-----|------|------|
| 游客口碑 | 差评如潮 | 口碑极佳 | 12% |
| 亲子友好度 | 不适合 | 亲子天堂 | 6% |
| 老人友好度 | 高强度 | 轻松舒适 | 5% |
| 夫妻友好度 | 无氛围 | 极致浪漫 | 8% |
| 一人游友好度 | 不便 | 自在 | 5% |
| 交通可达性 | 极不便 | 地铁直达 | 8% |
| 文化/生态/独特价值 | 无特色 | 世界级 | 10% |
| 基础设施与配套 | 简陋 | 一流 | 10% |
| **性价比** | 贵且不值 | 超高性价比 | **7%** |
| **旅游安全度** | 高风险 | 极安全 | **10%** |
| **华人安全度** | 歧视风险 | 非常友好 | **10%** |

| 综合加权满意度 | 以上指标按权重加权计算 |
| 备注 | 一句话描述这个地方 |

### 口碑方差解读

- **0-1.5 (绿)** = 评价稳定，去了基本不失望
- **1.5-2.5 (黄)** = 多数满意，少数不同意见
- **2.5-5 (红)** = 两极分化，体验看运气/时间/期望
