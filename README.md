## 统一列 (32 列)

所有 CSV 使用相同列结构，评分均为 **0-10 分制**：

| 指标 | 说明 |
|------|------|
| 游客口碑 | 综合好评度 |
| 亲子友好度 | 带娃适合度 |
| **适合儿童年龄段** | 如 "2-14岁"、"全年龄" |
| 老人友好度 | 步行量/无障碍/节奏 |
| 夫妻友好度 | 浪漫度/私密度 |
| 一人游友好度 | 独行体验 |
| 交通可达性 | 从主要城市到达便捷度 |
| 文化/生态/独特价值 | 文化底蕴与独特性 |
| 基础设施与配套 | 餐饮/住宿/wifi 等 |
| 性价比 | 体验 vs 花费 |
| **旅游安全度** | 当地治安与自然灾害风险 |
| **华人安全度** | 对华人友好度 |
| 综合加权满意度 | 以上指标按权重计算 |

## 数据来源

- 文化和旅游部官方 5A 名单
- 携程/马蜂窝/大众点评综合口碑
- Claude + GPT 交叉评估，1/2 加权平均
- 各国旅游局公开数据
